Nighthawk: AI/LLM Implementation Security Scanner

NightHawk is an open-source security analysis tool focused on AI and LLM application risks.

üìã Overview

NightHawk is an open-source security analysis tool designed for applications using AI and LLMs. It scans local and GitHub projects to detect AI-specific risks including prompt injection, unsafe tool invocation, secret exposure, and insecure data flows. Combining rule-based scanning with LLM reasoning, it generates structured JSON and Markdown reports detailing issues, affected files, exploit scenarios, and remediation steps.
